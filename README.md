keyboard commands
=================
normal mode
-----------
Tab  - edit cell to the right
CR   - edit cell below
jklm - move cursor
JKLM - move cursor faster
i    - edit cell at start of cell
a    - edit cell at end of cell
C    - edit cell, delete old content

insert mode
===========
home - move cursor to pos 0
end  - move cursor to end of edit text
